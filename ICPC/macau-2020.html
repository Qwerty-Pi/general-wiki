Macau Regional 2020

A. Given array $A$ of length $N$. Calculate $\mathbb{E}[A_1 + A_1 \cdot A_2 + \cdots + A_1 \cdot A_2 \cdots A_N]$ in $O(N \log^2 N)$.

B. Given string $S$ and $N$ strings $T_i$ of length $M$ ($N \times M \le 10^4$). Operation: append character to $S$ according to probability. Find $\mathbb{E}[\text{Length of } S]$ for each prefix of a string $R$ ($1 \le |R| \le 10^4$). Reduce Unknown in Markov Chain.

C. Partition $\{w_i\}$ to $A \cup B$ such that minimum value of $w_x \oplus w_y$ where $x, y$ in the same group is maximised in $O(N \log N)$.

D. E = ATK × (1 − (Crit Rate)) + ATK × (1 + (Crit DMG Rate)) × (Crit Rate)

E. Given polyline $(0, 0) - (1, H_1) - \cdots - (N, H_N) - (N + 1, 0)$. Take photo at $(i, H_i)$ and spans $2W \times 2H$ ($W \le 5$). Maximise total area covered for each $K$ photos taken ($1 \le K \le N$) in $O(N^2 \cdot 2^{2W - 1})$.

F. Given $N$ nodes and each node has degree $D$. Construct a graph that has $C$ connected components.

G. Given array of $N$ values $A_i$. $i \rightarrow j$ iff $A_i \oplus A_j$ is power of 2. Supporting (1) appending value and (2) predict winner if starts at $k$, in $A \log A$.

H. Given $N$ ($\le 1000$) different spacecrafts of fuel $d_i$ at each lattice point (except $M$ ($\le 1000$) points), which support $v = (dx, dy)$ where $||v|| \le d_i$. Find number of ways to go from $(0, 0)$ to $(1000, 1000)$. Compute $\frac{1}{1 - G(x, y)} \mod x^{1001} \cdot y^{1001}$.

I. NIM Game which supports $N$ ($\le 4 \times 10^4$) updates of (1) Add pile of $A_i$ ($\lt 2^{15}$) stones and cost $B_i$ and (2) Delete rightmost pile. Find min cost to make XOR Sum of $A_i$ be $0$ with memory complexity $O(A \log N)$.

J. Given $N$ jewellery of colour $C_i$ and value $V_i$. Support $M$ operations of (1) Point update of jewellery and (2) Maximise total value if starts at $s$, skip at most $K$ ($\le 10$) jewellery and all jewellery distinct colour in $O(K \log N)$.

K. Given $N$ ($\le 50000$) posters of size $W \times H$ ($W, H \le 2000$) and bottom-left corner $(X_i, Y_i)$ to be posted between day $L_i$ and $R_i$ ($L_i, R_i \le 2000$). $M$ ($0 \le M \le 10^5$) pairs of posters cannot both reject. Construct configuration in $O(\frac{N^2}{w} + M)$.

L. Given an random array $A$ of length $N$ ($\le 50$) where $A_i = U[[1, n]]$. Find expected number of permutation $P$ such that $P_i \le A_i$ for all $i$.
