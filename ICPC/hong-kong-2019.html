Hong Kong Regional 2019

A. Given $N$ rectangles. Determine axes of symmetry.

B. Given a binary tree and can only remove perfect binary tree. Two player, remove all wins.

C. Given a tree where node $i$ has value $A_i$. Determine number of paths such that the values can form lengths of polygon in $O(n \log^2 n)$.

D. Find $N$ in base $K$.

E. Given array of $N$ integers where $N$ odd. Each operation chooses $3$ consecutive values and remains median. Find all possible last values in $O(N^2)$. $0/1$.

F. Dropping $N \le 1000$ Cube, Sphere and Regular Tetrahedron. Glued after contact. Determine height. ???

G. Construct a tree with weighted node such that there are $K$ ($\le 10^9$) distinct optimal solutions by pay cost to subset of nodes which covers all leaves.

H. Given array $H$ of $N$ integers which initially empty. Handle $M$ queries, (1) Set value to $H_i$ or (2) Find value of $\min_{L \le i \le R} | H_i - h |$ in fast $O(N \log N + M \log^2 N)$.

I. Given $N$ objects and $M$ events: (1) Add member observing $K$ ($1 \le K \le 3$) objects, and (2) All member observing certain object gain $y$, and output every member reaching goal, online in $O(n \log n \log V)$.

J. Define $f(x) = \sum_{1 \le i \lt j \le k} x_i \cdot x_j$. Count number of $x$ such that $x \equiv f(x) \pmod{M} (M \le 60)$ where $L \le x \le R \lt 10^{5000}$.

K. Given array $D$ of length $N$ ($\le 800$) and $M$ ($\le 50000$) $A$ and $B$ nodes with cost $C_i$. Find min-cost matching for each $1 \le K \le M$ matches. $O(NM)$ flow.
