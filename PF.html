Primality Test

Problem: Given $n \in \mathbb{Z}^+$. Determine if $n$ prime.

Miller-Rabin Test

Given $n - 1 = 2^s d$, then if $n$ prime, $a^{n - 1} - 1 = (a^d - 1)(a^d + 1) \cdots (a^{2^{s - 1} d} + 1) \equiv 0 \pmod{n}$.

Strong Pseudoprime to base $a$: $a^d \equiv 1 \pmod{n}$ or $a^{d \cdot 2^r} \equiv 1 \pmod{n}$ for some $0 \le r \lt s$.

Algorithm: Test for different bases $1 \lt a \lt n$ for several rounds. Attack

Lucas Test

Let $P, Q \in \mathbb{Z}^+$ where $D := \sqrt{P^2 - 4Q} \not\in \mathbb{Z}$, $\varepsilon(n) = (D / n)$ and $\delta(n) = n - \varepsilon(n)$.

Define $\alpha, \beta$ be the roots to $X^2 - PX + Q$ in $\mathbb{Z}(\sqrt{D})$ and $U_k = \frac{\alpha^k - \beta^k}{\alpha - \beta}, V_k = \alpha^k + \beta^k$.

Then if $n$ prime, $\alpha^{\delta(n)} - \beta^{\delta(n)} \equiv 0 \pmod{n}$. Factorise the same way as Miller-Rabin.

Proof. If $D$ is square, then both $\alpha$ and $\beta$ are in $\mathbb{Z}_n$. Otherwise, $|| \frac{\alpha}{\beta} || = 1$.

Strong Lucas Pseudoprime: $\delta(n) = 2^kq$ then $n \mid U_q$ or $n \mid V_{2^iq}$ for some $0 \le i \lt k$. Attack
