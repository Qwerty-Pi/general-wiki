DSA / ECDSA

Parameters

Public Parameters
  • G: Base group. For DSA, G is $\mathbb{Z}_p^*$. For ECDSA, G is Elliptic Curve over $\mathbb{Z}_p$ ($p$ prime).
  • g: Element of G.
  • q: Order of g in G. Always prime.
Private Key
  • x: Satisfies $0 \lt x \lt q$.
Public Key
  • y: Generated by $y = g^x$.

Requirements

DSA

Parameter Size L/N (bit length of p/q): 1024/160, 2048/224, 2048/256, 3072/256

Hash Function H: SHA-2 or SHA-3

Source: FIPS 186-4. Removed in FIPS 186-5.

Methods

Sign(m):
    z = H(m)[:N]
    k = rand(1, q - 1)
    r = ([k] G) mod q
    s = k^-1 (z + x * r) mod q
    return (r, s)

Verify(m, (r, s)):
    z = H(m)[:N]
    assert r == ([zs^-1] G + [rs^-1] P) mod q

Notes: Implicitly converts point to integer by $G \mapsto (G)_x$.
