LFSR (Linear Feedback Shift Register)

Definition

Given a sequence $\{a_n\}_{n \in \mathbb{N}}$ where $a_n \in F$ for all $n \in \mathbb{N}$. Suppose exists linear recurrence.

Minimal Polynomial: $g(x) = x^j + \sum_{i = 0}^{i = j - 1} b_i x^i$, where $a_{j + k} + b_{j - 1} a_{j - 1 + k} + \cdots + b_0 a_k = 0$ for all $k \in \mathbb{N}$.

Well-known Problems

Unknown Linear Recurrence

Problem: Given a sequence with unknown linear recurrence. Find the linear recurrence.

Solution: Berlekamp-Massey Algorithm (sage). Run in $O(n^2)$ and require $2 \, \deg \, g$ terms.

Compute k-th Term

Problem: Given linear recurrence $(\{a_i\}_{0 \le i \lt j}, g)$. Calculate $a_k$.

Solution: Calculate $x^k \mod g$ and dot product with $\{a_i\}_{0 \le i \lt j}$.

Location of Term

Problem: Given linear recurrence $(\{a_i\}_{0 \le i \lt j}, g)$ and $\{a_{k + i}\}_{0 \le i \lt j}$ for some $k$. Reverse $k$.

Solution: Diagonalisation or Jordan Canonical Form (Wiki). Reduce it to Integral Discrete Logarithm.
