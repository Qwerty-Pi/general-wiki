National Yang Ming Chiao Tung University 2025 Team Selection Programming Contest

A: Given $N$ points $A_i$. Find smallest angle from points (above all $A_i$) that contains all points $A_L$ to $A_R$. $O((N + Q) \log^2{N})$.

B: Given valid ranges which is monotonic for fixed $l$. Find number of binary palidromes constructed from validity of partition. $O(N^2 \log N)$.

C: Given three initial points and final points. Output if can transform in $O(1)$, preseving an angle in movement. Circumcircle and collinear point order.

D: Given a tree. Find union of two paths in $O(\log N)$.

E: Given $N$ points $A_i$. Find $\sum_{|A_i - A_j| \in (L, R]} |A_i - A_j|^k$ for $k = 0$ or $1$ for each $i$ in $O((N + Q) \log N)$.

F: Count number of string of length $N$ that consists of first $K$ letters only and does not contain certain submask of letters. $O(2^K \cdot (K + \log N))$.

G: Given a simple bipartite graph of $N$ ($\le 100$) nodes. Orient edges such that the sum of shortest distance to sink is minimised. Flow.

H: Given integer set $T$ of size $N$ ($\le 10^5$), find $\gcd_{S \subset T\\|S| = k} \text{sum } S$ for each $k$.

I: Given a functional graph. Find distance between $Q$ pairs of nodes.

J: Match $\{A_i\}$ and $[[N]]$ such that each pair forms perfect square. $N \le 1000$.

K: Trivial array calculation.

L: Count # of 1 in array.

M: Given array $A$ of length $N$ ($\le 2 \times 10^5$). Operation: replace $A_i$ by lower median of $A \setminus \{A_i\}$. Random select $i$, find expected time till every element equal.
