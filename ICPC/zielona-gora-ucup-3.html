Universal Cup 3 Stage 2: Zielona Gora

A: Given a DAG. Find sequence of paths from node $1$ to node $N$ such that each new path contains at least one edge not appeared.

B: Given a tree of some tokens. Move to adjacent cost $1$, and must move towards the root. Find cost for all nodes as root.

C: Given a $N \times N$ board with numbers as cost. Cover the board by $N \times N$ squares, minimising costs.

D: Given an array of length $N$. Find sum over all partitions, the product of range XOR.

E: Find matrix where $A_{i, j} \ge A_{i+1, j}, A_{i, j+1}$ and sum of $|A_{x, y} - z|$ is minimised. 0/1 and Divide-Conquer.

F: Find shortest substring of fibonacci string that contains given subsequence. $O(N \log N)$.

G: Two cyclic binary string of length $N$ given. Operation: swap length-$K$ substrings. Make strings monochromatic in $N$ operations.

H: Partition array to maximise the minimum average value.

I: Given a chain of $N$ nodes. Each node contains $(S_i, E_i)$ and each edge contains several pairs of $(s_i, e_i)$ to pick one. Find rightmost of prefix such that $A \cdot S + B \cdot E \ge C$. Minkowski Sum.

J: Given $X_i = U(0, 2^{a_i})$. Find probability where a non-degenerate polygon can be formed. Disjoint of fail cases.

K: Count the number of partition of array $A$ with value at index $i$ being $2^{A_i}$, such that each sum of set is power of $2$. Divide and Conquer.

L: Given $N$ chords on circle. Find maximum subset such that no two intersects. $O(n \cdot \texttt{ans})$

M: Define $f(\sigma) := \sum |\sigma_i - i|$. Find the $K$-th lexicographically smallest permutation with $f = b$. $O(n^6)$ modelling.
