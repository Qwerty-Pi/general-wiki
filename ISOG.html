SIDH (Supersingular Isogeny Diffie-Hellman)

Public Parameters: Supersingular curve $E_0$, torsion bases $E[2^{e_A}] = (P_A, Q_A)$ and $E[3^{e_B}] = (P_B, Q_B)$

Public Key: Compute isogeny $\phi_A : E_0 \rightarrow E_A$ generated by kernel $K_A = P_A + [s_A] Q_A$. Sends $E_A, \phi_A(P_A), \phi_A(Q_A)$.

Secret Key: $s_A \in \mathbb{Z}_{2^{e_A}}$.

Shared Secret: Compute isogeny $\phi_{SB} : E_A \rightarrow E_{SB}$ generated by kernel $K_{SB} = \phi_A(P_B + [s_B] Q_B)$. Shared Secret is $j(K_{SB}).$

CSIDH (Commutative Supersingular Isogeny Diffie-Hellman)
